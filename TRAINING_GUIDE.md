# The Exotica Agonda - Training Guide 🎓

This training guide is designed to help new staff members quickly learn how to operate the Admin Panel.

## 1. Daily Operations Checklist
- **Morning Shift (8:00 AM)**
  - Check the Dashboard for **Pending Inquiries**.
  - Review **Today's Bookings** (Expected Check-ins and Check-outs).
  - Verify room availability statuses in the **Rooms** tab.
- **Afternoon Shift (2:00 PM)**
  - Respond to any new Inquiries.
  - Finalize check-ins and update booking statuses to "Confirmed" or "Completed".
- **Evening Shift (8:00 PM)**
  - Export the Daily Report via the Dashboard.
  - Note down any maintenance requests or unavailable rooms.

## 2. Handling Booking Scenarios
### Scenario A: Walk-in Guest
1. Go to **Bookings** -> **Create Manual Booking**.
2. Ask for the guest's ID and contact number.
3. Check **Rooms** tab to confirm live availability.
4. Fill in the dates, save, and mark as **Confirmed**.

### Scenario B: Cancellation Request
1. Locate the booking via the Search bar in the **Bookings** tab using the Booking ID.
2. Click the Eye icon to view details.
3. Change the status to **Cancelled**.
4. (Optional) Process refund according to policy in the payment gateway.

## 3. Best Practices
- Never delete an Inquiry before replying. Always mark it as "Replied".
- Check the `next/bundle-analyzer` output periodically if you are an IT staff member handling performance.
- When adding new Rooms, ensure the images are properly compressed before uploading to save bandwidth.
