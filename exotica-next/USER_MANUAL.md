# The Exotica Agonda - Admin User Manual 🛡️

Welcome to the administrative portal of The Exotica Agonda. This guide will help you manage your resort effectively.

## 1. Dashboard 📊
The Dashboard provides an overview of your resort's performance.
- **Stats Cards**: View total bookings, active rooms, pending inquiries, and revenue at a glance.
- **Recent Bookings**: A quick view of the latest guest activities.
- **Export Report**: Click "Export Report" to download a CSV file of all recent booking data for your records.

## 2. Managing Bookings 📅
Navigate to the **Bookings** tab to see all reservations.
- **Search**: Find guests by name or Booking ID.
- **Status Filter**: View only "Confirmed", "Pending", or "Cancelled" bookings.
- **Details**: Click the "View" icon (eye) to see guest details, room type, and dates.
- **Confirm/Cancel**: Inside the details modal, you can update the booking status.

## 3. Room Management 🏠
The **Rooms** tab allows you to control your room inventory.
- **Add Room**: Click "Add New Room" to create a new entry with image, price, and category.
- **Edit**: Update prices or room details using the pencil icon.
- **Delete**: Remove rooms that are no longer available.
- **Status Toggle**: Easily switch a room between "Available" and "Occupied".

## 4. Inquiries ✉️
Interact with prospective guests in the **Inquiries** section.
- **Mark as Replied**: Keep track of your communications.
- **Delete**: Clear old or spam inquiries.

## 5. Settings ⚙️
- **User Management**: Add or remove staff members who have access to this admin panel.
- **System Health**: Monitor API status and system backups.
- **Preferences**: Toggle "Email Alerts" and "Maintenance Mode".
